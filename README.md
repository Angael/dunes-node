# dunes-node
NPM library for easy video compression.

Uses `ffmpeg` and `ffprobe`

## Workflow

1. Write code
2. Commit & Push
3. `npm version patch`
4. `npm publish`

> Use `yarn link` to test this library locally in other projects before publishing
